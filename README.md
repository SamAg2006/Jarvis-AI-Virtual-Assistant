# ⚙️ INSTALLATION GUIDE  

## 1️⃣ Clone the Repository  

```bash
git clone https://github.com/SamAg2006/Jarvis-AI-Virtual-Assistant.git
cd Jarvis-AI-Virtual-Assistant
```

---

## 2️⃣ Install Required Libraries  

```bash
pip install speechrecognition pyttsx3 requests pyaudio
```

---

## 3️⃣ Install Ollama  

🔗 Download Ollama:  
https://ollama.com/

---

## 4️⃣ Run TinyLlama Model  

```bash
ollama run tinyllama
```

> ⚠️ Keep this terminal running in the background.

---

## 5️⃣ Launch Jarvis GUI  

```bash
python gui.py
```
